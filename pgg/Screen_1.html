{{ block title }} 第一個決策項目 {{ endblock }} {{ block content }}

為確保您對第一個決策項目的任務及報酬理解正確，請仔細閱讀此實驗說明。
下一頁將有一理解測驗，正確回答後您可進行第一個決策項目。

此決策項目中有兩種角色，「甲」與「乙」。每場實驗的受試者可能為「甲」，也可能為「乙」。
兩種角色的任務如下。

  • 「甲」的任務是選填以下 {{ Constants.num_quest }} 項中的 {{ Constants.num_quest_need }} 項資料。完成後，「甲」可自第一個決策項目獲得 {{ Constants.a_role_payoff }} 元。
    「甲」選填的答案將以匿名的方式存取，但填寫的每個答案都可能被驗證。每個問題的驗證方式將於點擊該問題時顯示在問題的下方。
    「甲」須將答案填寫在問題後方的空格。在這場實驗的最後，電腦會自每 {{Constants.a_role_val_num}} 名「甲」中隨機抽取一名驗證所有答案。若答案與驗證方式查詢到的一致，被驗證的「甲」將額外獲得 {{Constants.a_role_bonus}} 元。若任何一題驗證時發現錯誤，被驗證的「甲」由第一個決策項目獲得的報酬將全數歸零。為極大化實驗報酬，若您為「甲」，請照實填寫，亦可參考驗證方式查詢正確答案。
    照實填寫將極大化「甲」的實驗報酬，但並不代表「甲」同意此部分的答案於未來實驗或本實驗中使用。實驗結束前，「甲」可於一頁面選擇是否授權實驗者於未來實驗中使用此部分的匿名資料。
  • 「乙」的任務是使用其他場次中的某「甲」已授權使用的 {{Constants.num_quest_need}} 項資料。每個問題的資料將於點擊該問題時顯示在問題的下方。「乙」須將資料複製貼上至問題後方的空格。完成 {{Constants.num_quest_need}} 項資料後，「乙」可自第一個決策項目獲得 {{Constants.b_role_payoff}} 元。
    在這場實驗的最後，電腦會自每 {{Constants.b_role_val_num}} 名「乙」中隨機抽取一名驗證所有答案。若答案與某「甲」選填的資料一致，被驗證的「乙」將額外獲得 {{Constants.b_role_bonus}} 元。若任何一題驗證時發現錯誤，被驗證的「乙」由第一個決策項目獲得的報酬將全數歸零。為極大化實驗報酬，若您為「乙」，請完整複製貼上。
    完整複製貼上將極大化「乙」的實驗報酬。「乙」使用的資料為其他場次中的某「甲」已授權同意實驗者使用的 {{Constants.num_quest_need}} 項資料。
    閱讀完畢後，請點擊「下一頁」繼續本實驗。
